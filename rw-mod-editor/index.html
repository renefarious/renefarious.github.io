RW Mod Editor

Active

by Renefarious Studios

Android Free v1.0.0

RW Mod Editor makes it easy to create and edit unit mods for Rusted Warfare on your Android device. Open a mod file, make your changes, and save — all without needing to type raw code.

Download on Google Play Contact Support

Features

What you can do.

Everything you need to mod Rusted Warfare units, all in one place.

Open and edit mod files

Load any Rusted Warfare unit file from your device and edit it with a simple, organised layout.

Smart property hints

As you type a property name, the app shows matching suggestions and explains what each one does.

Colour picker

Pick colours from a visual selector with live preview before you save.

Start from a template

New to modding? Use a ready-made unit or building template to get started quickly.

Custom fonts

Adjust text size and font to suit how you like to read and work.

Works on most devices

Compatible with Android 5.0 and newer — covers the vast majority of phones and tablets.
