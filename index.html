RW Mod Editor

Material 3 INI editor for Rusted Warfare unit modding

A full-featured Android app for creating and editing Rusted Warfare unit definition files (.ini). Designed with Material 3 components and animations, it offers drag-to-reorder sections, property autocomplete with descriptions, a colour picker, template system, font customisation, Firebase Crashlytics error reporting, and a built-in property library covering core, turret, movement, graphics, AI, animation, effects, and more.
